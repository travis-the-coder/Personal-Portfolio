Can I sort them?

Hello, I have created this page solely for the purpose to demonstrate what bubble sort is and what it can do.

Bubble Sort is the ability to sort an array of numbers[in this case] and be able to compare them to each other and see which on is greater than the other. If the number happens to be greater than the next one, they will switch spots. The algorithm will keep doing this until all of the numbers are in order.
